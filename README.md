# Lagac

Lagac (Like-a-groq-ai-wrapper) は、Groq AI API を利用するためのコマンドラインインターフェース (CLI) クライアントです。

## 特徴

- **Groq モデルとのチャット**: 利用可能なモデルから自動的に最適なものを選択します (Llama 3, Mixtral などを優先)。
- **コマンド実行**: AI が提案したシェルコマンド (Windows では PowerShell、Linux では Bash) を実行できます。
  - 実行前に確認プロンプトが表示されます。
  - `y` (実行), `n` (スキップ), `Y` (以降すべて実行), `N` (以降すべてスキップ) で制御可能です。
- **コンテキスト注入**:
  - **ファイル読み込み**: `@ファイルパス` と入力すると、そのファイルの内容をプロンプトに追加します。
  - **コマンド出力**: `>コマンド` と入力すると、そのコマンドを実行し、出力をプロンプトに追加します。
- **API キー管理**: Groq API キーを `settings.json` に保存して管理します。

## セットアップ

1. .NET SDK を使用してアプリケーションをビルド・実行します。
2. 初回起動時に Groq API Key の入力を求められます。
   - API Key は Groq Console から取得できます。

## 使い方

アプリケーションを起動します:

```bash
dotnet run
```

### チャット操作

- メッセージを入力して Enter を押すと AI と会話できます。
- **ファイルの内容を含める**: `@Program.cs` のように入力します (ファイルサイズ制限あり)。
- **コマンドの実行結果を含める**: `>git status` のように入力します。
- **終了**: 空行を入力するとチャットを終了します。

## 設定

設定は `settings.json` に保存されます。`maxFileSize` で読み込み可能な最大ファイルサイズを指定できます。
